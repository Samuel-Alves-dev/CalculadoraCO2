# 🌱 Calculadora de CO₂

Este é um projeto simples de uma *Calculadora de Emissões de CO₂, desenvolvido com **HTML, CSS e JavaScript*. O objetivo é estimar a quantidade de dióxido de carbono gerada por atividades do dia a dia, como transporte e consumo de energia.

O projeto foi desenvolvido com auxílio do *GitHub Copilot*, utilizando sugestões de código para acelerar o desenvolvimento e melhorar a estrutura da aplicação.

---

## 📌 Objetivo

Ajudar o usuário a entender o impacto ambiental das suas ações, promovendo consciência sobre a emissão de gases de efeito estufa de forma simples e interativa.

---

## ⚙️ Funcionalidades

- Cálculo de emissões de CO₂ em tempo real  
- Entrada de dados pelo usuário (distância percorrida e tipo de transporte)  
- Exibição do resultado diretamente na tela  
- Interface simples e responsiva  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- GitHub Copilot (assistente de desenvolvimento)

---

## 🚀 Como executar o projeto

1. Clone este repositório:

git clone https://github.com/seu-usuario/calculadora-co2.git

2. Acesse a pasta do projeto:

cd calculadora-co2

3. Abra o arquivo index.html no navegador:

- Dê um duplo clique no arquivo  
- Ou use a extensão Live Server no VS Code  

---

## 📊 Exemplo de uso

Digite a distância percorrida de carro: 20 km  
Emissão estimada de CO₂: 4.6 kg
